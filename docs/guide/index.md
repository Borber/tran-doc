# 简介

`Tran` 的定位是一个跨桌面平台的简洁, 快速, 划词翻译软件.

## 项目目标

-   跨平台
-   简洁
-   快速

### 非目标

-   OCR
-   移动端
-   多翻译源
-   文件翻译
-   翻译面板
-   历史记录
-   同步功能

> 我将在后面的文档中详细介绍其中部分原因
